# Practică FSEGA - peviitor.ro

Bine ai venit! Acesta este proiectul tău de practică pentru FSEGA Cluj-Napoca.

---

## Ce trebuie să faci

### 1. Fă fork la acest repo
Apasă butonul **"Fork"** din dreapta sus pe GitHub pentru a crea o copie a proiectului pe contul tău.

### 2. Clonează repo-ul
```bash
git clone https://github.com/TASTERUL_TAU/practica-fsega.git
cd practica-fsega
```

### 3. Configurează Git
```bash
git config user.name "Numele Tău"
git config user.email "email@tau.ro"
```

### 4. Lucrează la task-uri
În GitHub Issues vei găsi toate task-urile organizate pe săptămâni:
- Fiecare issue are subtask-uri cu checkbox-uri
- Bifează task-urile pe care le-ai finalizat
- Start Date, End Date și Estimated time sunt setate pentru fiecare zi

### 5. Creează un branch pentru munca ta
```bash
git checkout -b nume-branch
```

### 6. Lucrează și commit
```bash
git add .
git commit -m "Descrierea lucrării tale"
```

### 7. Fa push și creează Pull Request
```bash
git push origin nume-branch
```
Apoi creează un Pull Request din branch-ul tău.

## Proiectul Final

**Tema:** Scrapuirea a 10 companii noi care nu sunt în prezent în peviitor.ro

**Cerințe:**
- Alege 10 companii care nu sunt scrapuite
- Creează un scraper pentru fiecare companie
- Proiectul trebuie să funcționeze local, fără internet
- Creează un Pull Request la final
