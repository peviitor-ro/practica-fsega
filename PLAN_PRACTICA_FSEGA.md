- **Tema**: Scrapuirea a 10 companii noi care nu sunt în prezent în peviitor.ro
- **Cerințe**: Minim 20h de lucru
- **Flux de lucru**: Fork → Clone → Work → Commit → Push → PR

- **Pași proiect**:
  1. Alegere 10 companii care nu sunt scrapuite
  2. Creare scraper pentru fiecare companie
  3. Integrare în sistemul peviitor.ro
  4. Testare și documentație

- **Deliverables**:
  - Fork propriu cu codul pe GitHub
  - README cu instrucțiuni
  - Link PR la repo principal
  - Prezentare 15-20 minute
